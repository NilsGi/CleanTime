Sidan räknar bakåt från dagens datum. Resultatet blir ett datum som kan sparas som clean_date i den befintliga tabellen.

Hur räknas det?

Beräkningen utgår från dagens datum och använder kalenderlogik: först dras år bort, sedan månader och sist dagar. På så sätt får du ett riktigt datum, inte ett uppskattat antal dagar.
